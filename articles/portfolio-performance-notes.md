# Portfolio Performance Notes

ポートフォリオを運用する中で、表示の重さに関するフィードバックが増えてきたため、フロント側の処理を中心に見直しました。

## 取り組んだこと

- スクロールイベントの処理を `requestAnimationFrame` ベースへ変更
- 画像読み込みをネイティブ `loading="lazy"` に統一
- 主要CDNへの `preconnect` を追加

## 改善の考え方

見た目を変えずに体感を改善するには、レイアウトや色を触るよりも、メインスレッドの負荷を減らすことが効果的でした。
特に、スクロール連動の処理を毎フレーム計算しないだけで、操作時の引っかかりがかなり減りました。

## 今後の改善候補

- 画像フォーマットの見直し（WebP/AVIF）
- Lighthouse計測の自動化
- 外部スクリプトの読込順最適化
