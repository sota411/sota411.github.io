### 自身のポートフォリオを公開しています

## 記事運用（Markdown）

1. `npm run add-article` を実行し、対話形式で記事を追加します。
2. 作成された `articles/*.md` に本文を書きます。
3. 本文やメタ情報を更新した場合は `npm run build-articles` を実行します。

### index.json の必須項目

- `slug`（英小文字・数字・ハイフン）
- `title`
- `date`（`YYYY-MM-DD`）
- `summary`
- `source`（Markdownファイル名）
